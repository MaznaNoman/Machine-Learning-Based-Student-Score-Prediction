# Student Score Prediction

This machine learning project predicts student scores based on study hours.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Model
- Linear Regression

## Dataset
Student study hours and corresponding scores.

## Project Workflow
1. Data Loading
2. Data Visualization
3. Train-Test Split
4. Model Training
5. Prediction
6. Model Evaluation
